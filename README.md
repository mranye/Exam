#### 功能

实现了老师登录系统后，对学生信息进行增删改；学生登录系统后，核对信息，限时考试，查看成绩和错题的功能。

#### 运行方法

开启MySql服务，导入数据库，先运行服务端的ExamServer.jav文件，再运行客户端的ExamClient.java文件。
